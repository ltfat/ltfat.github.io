SPSI - Single Pass Spectrogram Inversion (SPSI) for real signals

Usage

c=spsi(s,a,M);
c=spsi(s,a,M,mask,phase);

Input parameters

s \(M2 \times N\) array of modulus of Gabor coefficients.
a Length of time shift.
M Number of channels.

Output parameters

chat \(M2 \times N\) array of coefficients with reconstructed phase.

Description

c=spsi(s,a,M) returns array of coefficients c with reconstructed frequency invariant phase (the default convention in dgtreal) such that the they can be directly used in idgtreal.

c=spsi(s,a,M,'timeinv') returns coefficients with time invariant phase. To reconstruct the signal function idgtreal must be called with 'timeinv' i.e:

fhat = idgtreal(...,'timeinv').

This code was downloaded from http://anclab.org/software/phaserecon/m-files.zip (on 16.9.2015). Original name of the function was spsi.m It has been modified to work on the Gabor coefficients directly and +/- pi alternation is no longer necessary.
