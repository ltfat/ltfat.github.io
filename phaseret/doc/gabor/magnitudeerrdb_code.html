MAGNITUDEERRDB - Computes spectral convergence in dB

Program code:

function Edb = magnitudeerrdb(target,reconstructed)
%MAGNITUDEERRDB Computes spectral convergence in dB
%   Usage: Edb = magnitudeerr(target,reconstructed)
%

%
