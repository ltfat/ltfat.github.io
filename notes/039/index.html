Designing tight filter bank frames for nonlinear frequency scales

Nicki Holighaus, Zdeněk Průša, Christoph Wiesmeyr: Designing tight filter bank frames for nonlinear frequency scales

The following achive ltfatnote039.zip contains scripts reproducing figures and tables from the paper.

The paper is available here.

Please note that there is now an extended version of this paper available here.

Please note that LTFAT toolbox (development version or version>=2.0.2) must be installed in order to run the scripts.

Due to the bug in version 2.0.2, the frame bound ratios computed by ltfatnote039_table1and2.m are too large. Please use version 2.1.3 or higher to obtain more precise (and lower) frame bound ratios. The updated script now computes the frame bounds for a higher range of redundancies.

It is also STRONGLY recommended (but not required) to use the compiled MEX/OCT functions.
