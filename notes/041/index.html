Reassignment and synchrosqueezing for general time-frequency filter banks, subsampling and processing

Nicki Holighaus, Zdeněk Průša, Peter L. Søndergaard: Reassignment and synchrosqueezing for general time-frequency filter banks, subsampling and processing
Reproducible research attendum to the paper. The following achive ltfatnote041.zip contains scripts reproducing figures and examples from the paper.
Please note that LTFAT toolbox (development version or version>=2.1.1) must be installed in order to run the scripts.
It is also STRONGLY recommended (but not required) to use the compiled MEX/OCT functions.
