LOWDISCREPANCY - Returns a low discrepancy sequence

Usage

delays=lowdiscrepancy(name)

Input parameters

name Name of the low discrepancy sequence

Output parameters

delays Anonymous function specifying the sequence

Description

lowdiscrepancy(name) returns a low discrepancy sequence for the usage as a delay generating function in conjunction with waveletfilters. Currently, a kronecker sequence and a digital net are implemented.
