WFILT_SYMTIGHT - Symmetric Nearly Shift-Invariant Tight Frame Wavelets

Usage

[h,g,a] = wfilt_symtight(K);

Description

[h,g,a]=wfilt_symtight(K) with \(K \in {1,2}\) returns 4-band symmetric nearly shift-invariant tight framelets.
