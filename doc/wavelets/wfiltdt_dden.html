WFILTDT_DDEN - Double-Density Dual-Tree DWT filters

Usage

[h,g,a] = wfiltdt_dden(N);

Description

[h,g,a]=wfiltdt_dden(N) with \(N \in {1,2}\) returns filters suitable for dual-tree double density complex wavelet transform.
