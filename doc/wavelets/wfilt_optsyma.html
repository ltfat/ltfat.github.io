WFILT_OPTSYMA - Optimizatized Symmetric Self-Hilbertian Filters

Usage

[h,g,a] = wfilt_optsyma(N);

Description

[h,g,a]=wfiltdt_optsyma(N) with \(N \in {1,2,3}\) returns filters suitable with optimized symmetry suitable for for dual-tree complex wavelet transform tree A.

Examples:

wfiltinfo('optsyma3');
