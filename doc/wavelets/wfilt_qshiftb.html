WFILT_QSHIFTB - Improved Orthogonality and Symmetry properties

Usage

[h,g,a] = wfilt_qshiftb(N);

Description

[h,g,a]=wfilt_qshiftb(N) with \(N \in {1,2,3,4,5,6,7}\) returns Kingsbury's Q-shift wavelet filters for tree B.

Examples:

figure(1);
wfiltinfo('qshiftb3');

References:

N. G. Kingsbury. A dual-tree complex wavelet transform with improved orthogonality and symmetry properties. In ICIP, pages 375--378, 2000. [ http ]

N. Kingsbury. Design of q-shift complex wavelets for image processing using frequency domain energy minimization. In Image Processing, 2003. ICIP 2003. Proceedings. 2003 International Conference on, volume 1, pages I--1013--16 vol.1, Sept 2003. [ DOI ]
