WFILT_SYMDS - Symmetric wavelets dyadic sibling

Usage

[h,g,a] = wfilt_symds(K);

Description

[h,g,a]=wfilt_symds(K) with \(K \in {1,2,3,4,5}\) returns symmetric dyadic sibling wavelet frame filters from the reference.

The returned filterbank has redundancy equal to 2 and it does not form a tight frame.

Examples:

wfiltinfo('ana:symds3');
