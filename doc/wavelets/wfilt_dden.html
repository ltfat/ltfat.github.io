WFILT_DDEN - Double-DENsity DWT filters (tight frame)

Usage

[h,g,a] = wfilt_dden(N);

Description

[h,g,a]=wfilt_dden(N) computes oversampled dyadic double-density DWT filters. The redundancy of the basic filterbank is equal to 1.5.

Examples:

wfiltinfo('dden5');
