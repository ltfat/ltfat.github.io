WFILT_DDENB - Double-Density Dual-Tree DWT filters

Usage

[h,g,a] = wfilt_ddenb(N);

Description

[h,g,a]=wfilt_ddenb(N) with \(N \in {1,2}\) returns filters suitable for dual-tree double density complex wavelet transform tree A.

Examples:

wfiltinfo('ddena1');
