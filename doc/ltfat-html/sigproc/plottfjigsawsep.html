PLOTTFJIGSAWSEP - Plots separated layers from tfjigsawsep

Usage

plottfjigsawsep(fplanes,cplanes,info);
plottfjigsawsep(fplanes,cplanes,info,fs);

Description

plottfjigsawsep(fplanes,cplanes,info) shows the original and the three separated layers in both the time and the time-domains. All parameters can be obtained from tfjigsawsep.

plottfjigsawsep(fplanes,cplanes,info,fs) works as above but scales the axes labels according to the sampling rate fs.

The function calls plotdgtreal and forwards any arguments it understands.

Additional parameters

'equalyrange'
The y-axis range is equal in all time domain plots. By default, the individual y-axes are scaled automatically.
'showbuttons'
The figure will contain buttons for convenient playback of the separated layers. Note that the sampling rate fs must be provided. This is disabled by default and does not work on Octave.
