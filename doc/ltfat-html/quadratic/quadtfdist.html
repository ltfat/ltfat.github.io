QUADTFDIST - Quadratic time-frequency distribution

Usage

p = quadtfdist(f, q);

Input parameters

f Input vector:w
q Kernel

Output parameters

p Quadratic time-frequency distribution

Description

For an input vector of length L, the kernel should be a L x L matrix. quadtfdist(f, q); computes a discrete quadratic time-frequency distribution.
