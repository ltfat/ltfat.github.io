DEMO_DGT_PARAMETRIZE - shows how to parametrize the DGT algorithm

Description

This demo shows two ways of parametrizing the discrete Gabor transform. In the first part, it is shown how to parametrize the dgt to yield a specified signal length for a given window length and overlap. In the second part, the dgt algorithm is used to calculate a classical short-time Fourier transform with a = b = 1.
